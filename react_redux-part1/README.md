## 📝 Project Description

Introduction to **React Redux** - the official React bindings for Redux. Learn how to set up a Redux store, create actions and reducers, connect React components to the store using `connect` and `Provider`, and manage application state in a predictable way.

### 🎯 Learning Objectives

- Understand the Redux architecture (store, actions, reducers)
- Create and configure a Redux store
- Write action creators and action types
- Build reducers to handle state transitions
- Connect React components to the Redux store
- Use `Provider` to make the store available to the app
- Map state and dispatch to component props
- Write tests for actions and reducers
- Ensure no console errors or warnings
- Write maintainable, lint-free code

---

## 🔧 Key Concepts

- **Store** - Single source of truth for application state
- **Actions** - Plain objects describing what happened
- **Reducers** - Pure functions that specify state transitions
- **`connect`** - HOC to bind components to the store
- **`Provider`** - Makes the Redux store available to the component tree
